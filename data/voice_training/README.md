## Quick Start

### 1. Define Your Voice Goals

Edit `voice_profile.yaml` with your desired voice characteristics.

### 2. Collect Audio Samples

**Option A: Record yourself**
- Use Audacity or similar tool
- 50-100 clips, 5-15 seconds each
- Save as WAV files in `raw/` directory

**Option B: Find reference audio**
- Public domain audiobooks/podcasts
- Extract clips with desired style
- Save in `raw/` directory

### 3. Create Metadata

```bash
# Copy template
cp metadata_template.json metadata.json

# Edit metadata.json with your transcripts and labels
```

### 4. Prepare Dataset

```bash
# From project root
python src/training/prepare_voice_dataset.py
```

This will:
- Process all audio files (resample, normalize)
- Validate quality
- Create `processed/dataset.json` for training

### 5. Train Model

```bash
# Stop Ollama first (free VRAM)
ollama stop

# Train
python src/training/finetune_csm_voice.py
```

## Metadata Fields Explained

```json
{
  "audio_file": "clip_001.wav",        // Filename in raw/ folder
  "text": "Your transcript here",      // Exact words spoken
  "style": "friendly_greeting",        // Style label (your choice)
  "emotional_tone": 0.7,               // 0.0-1.0 (neutral to expressive)
  "quality_rating": 5,                 // 1-5 (subjective quality)
  "notes": "Optional notes"            // Any observations
}
```

### Style Label Examples

Use consistent labels that match your voice goals:
- `friendly_greeting`
- `professional_explanation`
- `empathetic_response`
- `enthusiastic_reaction`
- `thoughtful_consideration`
- `casual_conversation`
- `technical_explanation`

## Audio Quality Requirements

- **Format:** WAV (uncompressed) or FLAC
- **Sample rate:** 24kHz or 48kHz (will be resampled to 24kHz)
- **Channels:** Mono or stereo (will be converted to mono)
- **Duration:** 2-30 seconds per clip
- **Quality:** Clear speech, minimal background noise, no clipping

## Recording Tips

- Use a decent microphone (USB mic or headset)
- Record in a quiet room
- Maintain consistent distance from mic (6-12 inches)
- Speak naturally (don't read robotically)
- Stay hydrated (avoid mouth clicks)
- Record in short sessions (10-20 clips at a time)
- Review quality immediately after recording

## Dataset Size Recommendations

- **Minimal (20-50 samples):** Quick experiment, subtle adjustments
- **Good (100-200 samples):** Consistent voice characteristics, production quality
- **Excellent (500+ samples):** Professional quality, fine-grained control

Start with 50 samples and iterate based on results.
