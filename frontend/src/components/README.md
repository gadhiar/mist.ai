# Components

React components for the Mist.AI frontend.

## Planned Structure

- `ConversationDisplay/` - Main conversation display component
- `AudioStatus/` - Visual indicator for audio state (listening, speaking, etc.)
- `ConnectionStatus/` - WebSocket connection indicator
- `RichContent/` - Future: Display images, links, etc. from AI

## Component Guidelines

- Use TypeScript strict mode
- Use Tailwind CSS for styling
- Keep components small and focused
- Extract reusable logic into hooks
