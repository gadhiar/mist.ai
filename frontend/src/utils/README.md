# Utils

Utility functions for the Mist.AI frontend.

## Planned Utils

- `audioUtils.ts` - Audio format conversion, base64 encoding/decoding
- `formatters.ts` - Timestamp formatting, text formatting
- `constants.ts` - App-wide constants

## Utils Guidelines

- Pure functions when possible
- Document complex logic
- Add unit tests for critical utilities
- Use TypeScript for type safety
